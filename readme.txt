A kész produkciós verzióhoz csomagold ki a zengo_dist.rar-t, a Fejlesztői verzióhoz pedig csomagold ki a zengo.rar-t majd egy parancssorral navigálj oda ahova kicsomagolódott a projekt és futtass egy 'npm i'-t így feltelepülnek a szükséges csomagok a fejlesztéshez.
A weboldal vue.js alapú, bootstrap 4 keretrendszerrel és Sass Css kezelővel. A fejlesztéshez szükésges fájlok az index.html-en kívül az '/src' mappában találhatóak. Ha módosítani szeretnél .js vagy .scss fájlt akkor futtasd az 'npm run dev' parancsot, ez a szkript figyel a háttérben a módosulásokra és készít összevont fájlokat a megfelelő kiterjesztéssel amit a weboldal használ.
